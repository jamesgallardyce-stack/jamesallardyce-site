This is a lightweight static site built with Astro and designed to be hosted on Netlify.

## Local development
1. Install Node.js (version 20 recommended).
2. Install dependencies:
   - `npm install`
3. Run:
   - `npm run dev`

## Deploy to Netlify
- Connect this GitHub repository to Netlify.
- Build command: `npm run build`
- Publish directory: `dist`
